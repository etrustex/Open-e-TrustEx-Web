DESCRIPTION
    conf-env.cmd - helper script for configuring WebLogic domain on Windows OS
    conf-env.sh - helper script for configuring WebLogic domain on Unix family OS

USAGE
    conf-env.cmd <environment>
    conf-env.sh <environment>

    Possible environment values are:
        prod
        acc
        load
        test
        dev
        local

EXAMPLES
    conf-env.sh acc
    conf-env.cmd prod
    conf-env.cmd dev
